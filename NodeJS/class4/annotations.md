# Bases de datos
Colecciones organizadas de información. Diseñadas para gestionar de forma eficiente grandes volúmenes de datos, sean estructurados o no estructurados

## Relacionales (SQL)
Formato tabular (tablas) con filas y columnas. Columnas representan atributos / propiedades, filas son valores / entradas.

## No relacionales
Utilizan diversos modelos de datos. Para grandes volúmenes de datos, baja latencia, modelos de datos flexibles. <br>
Los datos se almacenan sin esquemas, sin las restricciones que requieren las relacionales.
